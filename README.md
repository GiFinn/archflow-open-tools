# ArchFlow Open Tools

ArchFlow Open Tools is an early-stage open-source toolkit for architecture students, young designers, and small studios.

The project helps structure repetitive parts of the architectural design process, such as room programs, material lists, model-building documentation, scale conversions, and early technical project notes.

## Why this project exists

Architecture projects often involve a lot of information that needs to be organized clearly before it can become a strong design. This includes room requirements, materials, model scales, documentation, and presentation texts.

ArchFlow Open Tools is not meant to replace design thinking. It is meant to support the workflow around the design process and make documentation more structured, reusable, and easier to understand.

## Current features

- Room program examples
- Material list templates
- Model-building workflow documentation
- Basic scale conversion script
- Structured project documentation examples

## Planned features

- Building-physics calculation helpers
- Model-scale conversion tools
- Room-program validation
- Automatic documentation exports
- Presentation summary templates
- More examples from real architecture student projects

## Target users

This project is mainly built for:

- Architecture students
- Young designers
- Small architecture studios
- People working on physical models and design documentation
- Beginners who want simple open-source tools for architectural workflows
