# Room Program Example

This example shows a simple room program for a small coffee bar project.

## Project type

Small coffee bar with indoor seating, service counter, storage, and outdoor terrace.

## Rooms and areas

| Area | Function | Notes |
|---|---|---|
| Entrance | Arrival and orientation | Clear view to counter |
| Guest area ground floor | Seating and circulation | Flexible tables |
| Counter | Ordering and service | Connected to preparation area |
| Preparation area | Coffee, food, storage access | Efficient workflow |
| Storage | Supplies and equipment | Close to service area |
| Outdoor terrace | Additional seating | Connected to guest area |
| Upper guest area | Quiet seating | More private atmosphere |

## Design notes

The room program helps organize the main functions before starting the detailed design. It can be used as a basis for diagrams, plans, model-building decisions, and presentation texts.
