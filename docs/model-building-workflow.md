# Model Building Workflow

This document describes a simple workflow for preparing an architectural model.

## 1. Understand the project

Before building the model, collect the most important project information:

- Project type
- Scale
- Main spaces
- Site context
- Materials
- Important design idea

## 2. Define the scale

Choose a scale that fits the task and the level of detail.

Common model scales:

- 1:500 for urban context
- 1:200 for building volume
- 1:100 for spatial organization
- 1:50 for detailed architecture
- 1:20 for interior or furniture-like details

## 3. Prepare the material list

Before starting, create a list of materials. This helps avoid mistakes and makes the process more efficient.

## 4. Build in layers

A clear model can be built in layers:

1. Base plate
2. Site context
3. Main volume
4. Openings
5. Interior structure
6. Details
7. Landscape or surroundings

## 5. Document the process

Take photos during the model-building process. These images can later be used for presentations, documentation, and project reflection.

## 6. Review

Before finishing the model, check:

- Is the scale correct?
- Are the main design ideas visible?
- Are the materials cleanly used?
- Is the model understandable without explanation?
